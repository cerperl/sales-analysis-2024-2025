# 零售销售数据分析项目

## 项目简介
本项目基于模拟的 2024-2025 年多店铺销售数据，完成了从 **数据清洗、聚合计算、移动平均、同比/环比分析** 到 **可视化与异常值检测** 的完整流程。

## 技术栈
- Python (Pandas, NumPy, Matplotlib)
- Jupyter Notebook

## 项目亮点
- 多维度销售与客单价分析
- 移动平均、同比/环比趋势计算
- 异常值检测与标记
- 可视化图表（趋势图、堆叠面积图、双轴对比图）
